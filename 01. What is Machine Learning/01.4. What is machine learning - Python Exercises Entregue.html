#Criar a base das matrizes nas variáveis para receber os dados
x = []
y = []
#print(x, y)

for n in range(2, 201): #intervalo de 2 a 201
    x.append(n) #O método append () adiciona um item ao final da lista.
    y.append(12+3*n+np.random.randint(10))    
#print(x, y)

plt.scatter(x, y)
plt.show()

x = np.array(x)
X = x[:,np.newaxis]

y = np.array(y)


model = LinearRegression(fit_intercept = True)
model.fit(X, y)

beta0 = model.intercept_
beta1 = model.coef_[0]

print('Posição yi = Intercepção β0 + Coeficiente β1 * Variável x_i + erro e_i')
print('Posição = ', round(beta0,2), ' + ', round(beta1,2), '* x_i + e_i\n')

print('Variância dos coêficientes: ', np.var(beta1))

Posição yi = Intercepção β0 + Coeficiente β1 * Variável x_i + erro e_i
Posição =  16.95  +  3.0 * x_i + e_i

Variância dos coêficientes:  0.0

Resposta:

  • Os efeitos de N de 2 a 200 é de uma correçalação positiva quase perfeita.
  • Visualmente quase não há dispersão dos ruídos, uma vez que as observações estão próximas à reta de regressão linear.
  • As estimativas dos coeficientes é estável, pois não apresenta variância.
